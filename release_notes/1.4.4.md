# Phiên bản 1.4.4

## Tính năng mới

### Settings Hub (Trung tâm Cài đặt)
- **Gộp cài đặt**: Tất cả các mục cài đặt được tập trung vào một màn hình duy nhất, giúp truy cập nhanh và dễ quản lý hơn.
- **Giao diện mới**: Thiết kế hiện đại với các section rõ ràng: Giao diện, Dữ liệu, Bảo mật, và Thông tin ứng dụng.

### Tùy chọn Font chữ
- **Chọn font linh hoạt**: Cho phép người dùng chọn font chữ yêu thích từ danh sách: UTM-HelvetIns, Inter, Roboto.
- **Xem trước font**: Hiển thị preview font ngay trong phần cài đặt trước khi áp dụng.
- **Lưu tùy chọn**: Font được chọn sẽ được lưu lại và áp dụng vào lần mở app tiếp theo.

## Cải tiến

### Tái cấu trúc thư mục
- **Tổ chức code tốt hơn**: Chia thư mục `lib/` thành các phần rõ ràng:
  - `core/`: Theme, utilities, extensions.
  - `data/`: Models, services, database.
  - `logic/`: Providers, state management.
  - `presentation/`: Screens, widgets, UI components.

### Giao diện
- Cải thiện side menu với icons và layout rõ ràng hơn.
- Sửa lỗi hiển thị và tối ưu UX cho các màn hình cài đặt.
