# Nhật ký thay đổi - Phiên bản 1.0.8+40

## Có gì mới
- **Quản lý Hồ sơ cá nhân (Citizen Profiles):** 
    - Cho phép lưu và quản lý danh sách hồ sơ (Mã số thuế, Biển số xe, CCCD).
    - Tự động ghi nhớ và chọn nhanh thông tin khi tra cứu.
    - Hỗ trợ thêm, sửa, xóa hồ sơ cá nhân linh hoạt.
- **Hoàn thiện Citizen Hub:** 
    - Đồng nhất giao diện tra cứu Thuế, Phạt nguội, BHXH.
    - Cải tiến tốc độ tải trang WebView.

## Kỹ thuật
- **Nâng cấp Database:** Chuyển đổi sang Schema v9 để hỗ trợ lưu trữ hồ sơ.
- **Tối ưu Build:** Cải tiến quy trình đóng gói ứng dụng, tận dụng tối đa RAM hệ thống.
- **Auto-check Update:** Cập nhật metadata để hệ thống báo update nhận diện bản 1.0.8.
