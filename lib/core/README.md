# Core Layer

This directory contains the essential foundation of the application, including shared constants, styling, and general utilities.

## Directory Structure

### `constants/`
- **`app_constants.dart`**: General application strings and configurations.
- **`color_constants.dart`**: Centralized color palette for the app.

### `theme/`
- **`app_theme.dart`**: Definition of the Material Theme (Dark/Light modes).

### `utils/`
- **`app_utils.dart`**: Generic helper functions (formatting, validation).

### `widgets/`
- High-level global widgets that are used across multiple feature domains.
